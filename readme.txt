React Basics
This repository contains several small projects that were completed as part of the "Advanced React" section of the Meta Professional Front End Dev Certificate. These projects cover various React concepts and techniques and are designed to help developers improve their understanding of React and how to use it in their own projects.

Getting Started
To get started with these projects, you will need to have Node.js and npm installed on your system. Once you have these installed, you can follow these steps:

Clone the repository to your local machine using git clone https://github.com/velaspis/react-basics.git.
Navigate to the project you want to work on, for example cd counter.
Install the project dependencies using npm install.
Start the development server using npm start.
You can now access the project in your browser at http://localhost:3000.

Projects
The following projects are included in this repository:

Counter
This project demonstrates the basics of React state management by building a simple counter that can be incremented and decremented.

Todo List
This project demonstrates how to build a simple todo list application using React. Users can add, edit, and delete items from the list.

Color Picker
This project demonstrates how to work with forms in React by building a color picker that allows users to select a color from a dropdown menu and see it displayed on the page.

Github Profile Viewer
This project demonstrates how to work with API calls in React by building an application that allows users to enter a Github username and see information about that user's profile.
